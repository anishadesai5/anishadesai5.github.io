Introduction

This project implements traditional recommendation systems for e-commerce businesses using machine learning techniques. The analysis uses the Amazon Electronics Ratings Dataset, containing approximately 7.8 million ratings across 4.2 million users and 476,002 products spanning from December 1998 to July 2014.

Baseline Models

Before diving into SVD, the project establishes foundational recommendation strategies:

  • Global Mean: Predicts the overall average rating (4.18)
  • User Bias: Adjusts predictions based on individual user rating tendencies
  • Item Bias: Accounts for products rated consistently higher or lower
  • Combined Bias: Incorporates both user and item biases simultaneously

    • The combined bias approach achieves the lowest Mean Absolute Error (0.8618), providing a benchmark for more advanced methods.

    SVD (Singular Value Decomposition)

    SVD is a matrix factorization technique that decomposes the sparse user-item rating matrix into latent factor representations. The core idea is to represent both users and items as vectors in a lower-dimensional latent space, where the dot product of these vectors approximates the rating.

    How SVD Works

    The prediction formula combines baseline components with latent factors:

    r̂ui = μ + bu + bi + puTqi

    Where:

  • μ is the global mean rating
  • bu is the user bias
  • bi is the item bias
  • pu is the user latent factor vector
  • qi is the item latent factor vector

    • Model Configuration

    The SVD model uses:

  • 20 latent factors
  • 20 training epochs
  • Learning rate: 0.005
  • Regularization: 0.02

    • Results

    SVD significantly outperforms the baseline models:

  • RMSE: 1.1134 (5.30% improvement over baselines)
  • MAE: 0.8412

    • SVD successfully captures hidden patterns in user preferences and item characteristics that simpler models miss. The latent factors learn to represent abstract concepts like product quality, user pickiness, or category preferences without explicit feature engineering.

    Key Findings

  • SVD provides meaningful improvements over simple bias-based models
  • All models struggle with predicting negative ratings due to dataset skew (55.6% are 5-star reviews)
  • Errors increase substantially for 1-2 star items, suggesting the need for additional features or alternative approaches for handling sparse negative feedback
